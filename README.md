## 简介
&emsp;&emsp;每个前端都应该有一个特效库，本项目原创或整理网上感觉不错的CSS3效果，致敬大神们。

## 文件结构
```shell
.
├── public  打包所需静态资源
└── src
    ├── untils  axios封装
    └── assets  项目静态资源
        ├── iconfont  自定义图标资源
        ├── img  图片资源
        ├── thumbnail  页面缩图
        └── css  全站样式
    ├── components  公共组件
    ├── home  欢迎页
    ├── fontEffect  文字效果
    ├── listEffect  列表效果
    ├── loadingEffect  加载效果
    ├── menuEffect  菜单效果
    ├── tableEffect  表格效果
    └── picEffect  图片效果
```
